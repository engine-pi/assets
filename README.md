# `assets` (Mediendateien)

Eine Sammlung von Mediendateien wie Bilder und Audio- und Videodateien

Da die Engine Pi in einem [Monorepo](https://en.wikipedia.org/wiki/Monorepo)
entwickelt wird, sind die Mediendateien in dieses Repository ausgelagert. Dieses
[Assets-Repository](https://github.com/engine-pi/assets) ist in dem
[Haupt-Repository](https://github.com/engine-pi/engine-pi) als
[Git-Submodule](https://git-scm.com/book/en/v2/Git-Tools-Submodules)
eingebunden.
